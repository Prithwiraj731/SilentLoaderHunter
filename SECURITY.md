# Security Policy

## Supported Versions

Currently supported versions:

| Version | Supported |
|----------|------------|
| 1.x      | ✅ Yes |
| < 1.0    | ❌ No |

---

## Reporting a Vulnerability

If you discover a security vulnerability in SilentLoaderHunter:

- Do NOT open a public issue.
- Report it responsibly via private message or email.
- Provide detailed reproduction steps.

Please include:
- Script version
- Windows version
- Steps to reproduce
- Expected vs actual behavior

---

## Responsible Disclosure

This project is intended for defensive and educational use only.

We encourage responsible disclosure practices and will investigate all legitimate reports.

---

Thank you for helping improve the security of this project.
